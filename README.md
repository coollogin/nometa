## 📋 Описание

**NoMeta** — это удобное десктопное приложение для удаления персональных метаданных из офисных документов. Программа помогает безопасно очищать информацию об авторе, датах создания/изменения и других персональных данных перед отправкой или публикацией файлов.

---

## 🎯 Возможности

| Функция | Описание |
|---------|----------|
| 🗂️ **Очистка метаданных** | Удаление автора, дат, компании, заголовков и других свойств |
| 📁 **Обработка папок** | Рекурсивный обход всех подпапок |
| 📄 **Множественные форматы** | Office, PDF, OpenDocument документы |
| 💾 **Безопасность** | Автоматический backup перед обработкой |
| 🔄 **Восстановление** | Автовосстановление файла при ошибке |
| 📊 **Прогресс** | Визуальная индикация обработки каждого файла |
| 🎨 **Современный UI** | Тёмная тема, минималистичный дизайн |

---

## 📂 Поддерживаемые форматы

| Программа | Форматы | Метод очистки |
|-----------|---------|---------------|
| Microsoft Word | `.doc`, `.docx` | Microsoft Office COM |
| Microsoft Excel | `.xls`, `.xlsx` | Microsoft Office COM |
| Microsoft PowerPoint | `.pptx` | Microsoft Office COM |
| PDF | `.pdf` | pdf-lib |
| OpenDocument | `.odt`, `.ods`, `.odp` | adm-zip |

---

## ⚙️ Требования

- **Операционная система:** Windows 7 и выше (x64)
- **Microsoft Office:** Требуется для обработки файлов Word, Excel, PowerPoint
- **Node.js:** 18.x (для сборки)

## 🚀 Установка

Скачайте установочный файл и запустите установку приложения.
Для Portable версии - просто скачайте exe файл и запустите его.

### Из исходников

```bash
# Клонирование репозитория
git clone https://github.com/ваш-логин/nometa.git

# Переход в директорию
cd nometa

# Установка зависимостей
npm install

# Запуск приложения
npm start
```

### Из архива

1. Скачайте `nometa.zip`
2. Распакуйте в любую папку
3. Установите Node.js 18.x
4. Выполните `npm install`
5. Запустите `npm start`


## 📖 Использование

### Выбор файлов

```
┌─────────────────────────────────────────┐
│              NoMeta                     │
├─────────────────────────────────────────┤
│                                         │
│     📁 Перетащите папку или файл        │
│                                         │
│     [Выбрать папку]  [Выбрать файлы]    │
│                                         │
├─────────────────────────────────────────┤
│  Найдено: 15 файлов                     │
│  Готово: 0                              │
│  Ошибки: 0                              │
├─────────────────────────────────────────┤
│         [Очистить метаданные]            │
└─────────────────────────────────────────┘
```

1. **Выберите папку** — нажмите кнопку или используйте `Ctrl+O`
2. **Выберите файлы** — нажмите кнопку для выбора отдельных файлов
3. **Начните обработку** — нажмите «Очистить метаданные» или `Enter`

### Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl+O` | Выбор папки |
| `Enter` | Начало обработки |
| `Escape` | Закрыть диалог |

## 🛡️ Безопасность

Приложение обеспечивает безопасную обработку файлов:

```
Оригинальный файл → Backup → Очистка копии → Проверка → Замена оригинала
         │                                        │
         └──────── Восстановление при ошибке ─────┘
```

- ✅ **Backup** — автоматическое создание резервной копии перед очисткой
- ✅ **Проверка целостности** — верификация файла после обработки
- ✅ **Автовосстановление** — возврат оригинала при любой ошибке
- ✅ **Безопасная замена** — атомарная операция замены файла
